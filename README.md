This repository contains the code and data for the paper "Observation of Cosmic-Ray Anisotropy in the Southern Hemisphere with 11 yr of IceTop Air Shower Array Data". It is organized by figure in the order listed in the paper. There are inline comments for ease of use.

Begin in the simulations or skymaps directories and follow the instructions in the directory READMEs to produce the figures. Then enter the directories for the figures you would like to produce and change the inputs to the notebooks there to the output files produced in previous steps. 

To produce the figures or tables from already existing data, enter the figure directories and run the code there without changing the input directories.
